# INFO2413_GROUPA Campus Marketplace App

A web-based campus marketplace platform designed for students and faculty members to buy, sell, and exchange used items safely within the school community.

## Project Overview

The Campus Marketplace App allows students to:
- List items for sale
- Browse available products
- Contact sellers
- Manage their own listings
- Create secure student accounts

The purpose of this project is to provide a simple and beginner-friendly marketplace system specifically for campus students.
